# Legacy Safety Lead Tracker

This project is ready to run as-is.

## Run locally
1. Install Node.js 18 or newer
2. In this folder, run:
   npm install
   npm run dev
3. Open http://localhost:3000

## Deploy on Vercel
1. Upload this folder to Vercel
2. Deploy
3. Open the URL on your iPhone in Safari
4. Tap Share > Add to Home Screen

## Notes
- Your logo is already included in /public/logo.png
- Leads are saved in the browser using localStorage
